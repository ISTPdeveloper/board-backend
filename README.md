# board-backend
### `django-rest-framework`을/를 이용한 게시판 만들기🎈

## Installation

```bash
$ pip install -r requirements.txt
```

## Start docker db

```bash
docker-compose up -d
```

## Setting ENV

```bash
$ cd config/settings

# Window OS
copy con base.py

# Mac OS
$ touch base.py
```

## Running the app

```bash
# development
$ python manage.py runserver
```
